Class: Mesh

tiled3d~Mesh(data)

Class representing a mesh: containing many triangles. This function may take an update function

Constructor

new Mesh(data)

construct a mesh. The constructor expects the data to have 3 constants keyword for it's modelling [ v, f, c]. v - vertices f - faces c - face color a valid data could be ` v 0 0 0 v 0 1 . v .... f 1 2 3 f 3 1 2 c 90 50 50` hsl
